# Chess System Java
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/Bergamos/chess-system-java/blob/master/LICENCE) 

# Sobre o projeto

Sistema completo de um jogo de xadrez rodando no terminal para aplicação dos conceitos aprendidos no curso de Java e POO do professor Nelio Alves.

Conteúdo aplicado: 
- Programação Orientada a Objetos.
- Estrtuturas sequenciais, condicionais e repetitivias.
- Sobrecarga.
- Listas e arrays.
- Herança, Encapsulamento e Polimorfismo.
- Tratamento de exceções.

# Tecnologias utilizadas
- Java

# Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/Begamos/chess-system-java

# entrar na pasta do projeto target/application
cd target/appplication

# executar o projeto
java Program.class
```
